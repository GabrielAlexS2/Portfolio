# SITE VISION: Gabriel Marques Editorial Portfolio

## 1. Core Concept
A high-end, editorial-style personal portfolio for a Senior/Lead Frontend Developer. The aesthetic blends Brutalism with sophisticated typography, relying heavily on stark contrast (Deep Onyx vs Vibrant Emerald) and meticulous grid layouts rather than traditional UI elements like cards or drop shadows.

## 2. Technical Stack
- Vanilla HTML5 / Semantic HTML
- Tailwind CSS via CDN (Utility-first styling, Container Queries, Forms)
- Custom CSS for typography scaling, outline text effects, and marquee animations
- Intersection Observer for scroll-based "reveal" animations
