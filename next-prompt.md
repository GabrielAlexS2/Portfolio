A minimalist but highly impactful Contact page acting as the final call-to-action for the portfolio.

**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Desktop-first
- Theme: Dark, editorial, high-contrast, minimalist
- Background: Deep Onyx (#0A0A0A)
- Primary Accent: Vibrant Emerald (#22C55E) for call-to-actions, hero sections, and highlights.
- Text Primary: Pure White (#FFFFFF)
- Text Secondary: Muted Platinum/Gray (rgba(255, 255, 255, 0.7) to text-gray-400) for body text that needs lower hierarchy.
- Typography Layout: Emphasize massive, bold Space Grotesk headers paired with clean Inter body text. Use uppercase with wide letter-spacing for small functional labels.
- Components: Pill-shaped buttons (fully rounded). Input fields should have incredibly minimal styling (bottom border only or very subtle gray background) to maintain the editorial feel.
- Structure: Use 12-column asymmetric grids. Maximize whitespace. Use fine 1px borders (border-gray-800) for separation instead of heavy drop shadows or card backgrounds.

**Page Structure:**
1. **Header:** Minimal navigation (matching the home page).
2. **Hero Section:** A massive typographic greeting ("VAMOS CONVERSAR?") and a brief subtext inviting networking or project inquiries.
3. **Contact Layout (2-Column Asymmetric):**
   - **Left Column (Details):** Email address in large Space Grotesk font. Below it, clean links for LinkedIn, GitHub, and Dribbble using uppercase tracking.
   - **Right Column (Form):** A stark, minimalist contact form (Name, Email, Message). Use inputs with only bottom borders to look like a high-end editorial fill-in-the-blank. Include a solid Emerald pill-shaped submit button ("Enviar Mensagem").
4. **Footer:** The massive scrolling marquee "GABRIEL MARQUES" header with social links and copyright (matching home page).
