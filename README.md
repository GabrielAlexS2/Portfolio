# 🚀 Portfólio Profissional - Gabriel Marques

Bem-vindo ao meu portfólio! Este projeto é uma vitrine das minhas habilidades como **Desenvolvedor Full Stack**, focado em criar experiências digitais de alto impacto, performance e design minimalista.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as melhores práticas de desenvolvimento web moderno:

- **Linguagens:** ![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)
- **Estilização:** ![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)
- **Animações:** ![GSAP](https://img.shields.io/badge/GSAP-88CE02?style=for-the-badge&logo=greensock&logoColor=white) (GreenSock Animation Platform)
- **Design:** Foco em **Dark Mode**, **Glassmorphism** e **Responsividade Total**.

## ✨ Funcionalidades em Destaque

- **Navegação Fluida:** Menu mobile intuitivo e transições suaves entre páginas.
- **Seção de Certificados:** Área dedicada e escalável para exibir conquistas acadêmicas e profissionais, com páginas de detalhes individuais.
- **Showcase de Projetos:** Exibição dinâmica de projetos reais, com destaque para o **MarketFlow** (Sistema de PDV e Estoque).
- **Animações de Scroll:** Efeito "Reveal on Scroll" para uma experiência de usuário imersiva.
- **Otimização SEO:** Estrutura semântica preparada para mecanismos de busca.

## 📂 Estrutura do Projeto

- `index.html`: Landing page principal com visão geral.
- `projects.html`: Galeria detalhada de projetos selecionados.
- `about.html`: Biografia, trajetória e conquistas.
- `certificates.html`: Hub de certificados e especializações.
- `skills.html`: Detalhamento de serviços e competências técnicas.
- `/css` & `/js`: Arquivos modulares de estilo e comportamento.
