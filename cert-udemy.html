Dev Web Completo

Carga Horária: 120h 20 Cursos + 20 Projetos

Este é um dos cursos mais abrangentes que concluí, totalizando **120 horas** de conteúdo prático e teórico. O curso **Desenvolvimento Web Completo** na Udemy abrange desde os fundamentos até tecnologias avançadas de mercado.

O grande diferencial desta formação foi a metodologia baseada em projetos: construí **20 aplicações reais**, o que me permitiu entender como integrar diferentes tecnologias em cenários de produção reais.

Tecnologias Dominadas:

  • HTML5, CSS3 e Design Responsivo.
  • JavaScript moderno (ES6+) e manipulação de DOM.
  • Desenvolvimento Backend com PHP e Node.js.
  • Bancos de Dados MySQL e integração de dados.
  • Segurança em aplicações web.
